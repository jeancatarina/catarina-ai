AI

AI That Understands Your Entire Project

Not another chatbot wrapper. Catarina AI gives Claude full context of your workspace — files, structure, dependencies — so every answer is grounded in your actual code.

  • Workspace-scoped conversations with full project context
  • Real-time streaming — see Claude think and code live
  • Extended thinking mode for complex architectural problems
  • Plan approval workflow — review before AI executes changes
  • Prompt enhancement — AI improves your prompts before sending
  • Model selection (Opus, Sonnet, Haiku) per conversation
  • Persistent chat history saved per project
🤖
Git

Git Without Leaving Your Flow

Branch management, diffs, and pull requests — all inside Catarina AI. Worktree-based isolation means you never stash again. Context switching is instant.

  • Worktree-based workspace isolation — no branch conflicts, ever
  • Real-time uncommitted + committed change tracking
  • Syntax-highlighted diff viewer with inline annotations
  • Pull request discovery across all your repos
  • PR status, CI checks, and review decisions at a glance
  • Create pull requests directly from the workspace
  • Auto-archive workspace when PR merges
🔀
Terminal

Full Terminals, Not Toy Shells

Real xterm.js terminals with full ANSI support. Run your build tools, test suites, and scripts without alt-tabbing. When something breaks, paste the error straight into AI.

  • Real xterm.js terminals with full ANSI color support
  • Multiple terminal tabs with custom color coding
  • Automatic working directory tracking
  • AI-powered command suggestions when you get stuck
  • Paste errors directly into AI for instant diagnosis
🖥
API

Test APIs Without Switching Apps

Ditch Postman. Catarina AI has a native HTTP request builder with collections, environments, and secret management baked in. SSRF protection keeps you safe by default.

  • HTTP request builder with method, URL, headers, and body
  • Request collections organized in drag-and-drop folders
  • Environment variables (Dev / Staging / Prod) with interpolation
  • Secret variables that never leak to logs or exports
  • Auto-detect framework and create environments instantly
  • Response viewer with syntax highlighting
  • SSRF protection built in — safe by default
📡
Focus

Stay Focused, Ship Faster

Productivity tools that live where you work. No more browser tabs for timers, notes, and planning. Everything stays in context, in the same window as your code.

  • Pomodoro timer with customizable work/break cycles
  • Day planner with categorized tasks and priorities
  • Quick notes tied to your active session
  • Full calendar with event creation and management
  • Excalidraw diagrams for architecture and whiteboarding
  • Link board for bookmarking developer resources

Rust

Built Different

Catarina AI is not another Electron app eating your RAM. It is a native Rust + Tauri 2 application with a tiny binary, instant startup, and near-zero idle memory. Your laptop fans will thank you.

  • Rust + Tauri 2 backend — native speed, tiny memory footprint
  • 15 languages supported out of the box
  • Cross-platform: macOS (ARM + Intel), Windows, Linux
  • Auto-updates with cryptographically signed packages
  • MCP server ecosystem for extensible tooling
  • Zero telemetry — all data stays local on your machine
