Installation

Quick setup guide for each platform.

🍎 macOS

  1. Download the .dmg file for your chip (Apple Silicon or Intel).
  2. Open the .dmg and drag Catarina AI to your Applications folder.
  3. On first launch, right-click the app and select Open to bypass Gatekeeper.

🐧 Linux

  1. AppImage: Make executable with chmod +x *.AppImage, then run.
  2. Debian/Ubuntu: Install with sudo dpkg -i catarina-ai_*.deb.
  3. Launch from your application menu or terminal.

💻 Windows

  1. Download the .exe or .msi installer.
  2. Run the installer and follow the setup wizard.
  3. If SmartScreen warns you, click More info then Run anyway.
