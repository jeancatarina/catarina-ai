Installation

Quick setup guide for each platform.

macOS

Recommended
brew tap catarina-claude/apps brew install --cask catarina-claude
or manual install
  1. Download the .dmg for your chip.
  2. Open and drag Catarina Claude to Applications.
  3. Right-click the app → Open to bypass Gatekeeper.

Linux

  1. AppImage: Make executable with chmod +x *.AppImage, then run.
  2. Debian/Ubuntu: Install with sudo dpkg -i catarina-claude_*.deb.
  3. Launch from your application menu or terminal.

Windows

  1. Download the .exe or .msi installer.
  2. Run the installer and follow the setup wizard.
  3. If SmartScreen warns you, click More info then Run anyway.
